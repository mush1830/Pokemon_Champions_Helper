# PokeOCR

포켓몬 챔피언스 화면을 실시간으로 인식해서 스피드 비교와 타입 약점을 오버레이로 표시해주는 도구입니다.

---

## 요구사항

- **Windows 10 / 11**
- **NVIDIA GPU** (VRAM 4GB 이상 권장)
- **인터넷 연결** (첫 실행 시 AI 모델 약 5GB 다운로드)
- 15~20GB 디스크 여유공간
---

## 설치

1. 이 폴더를 원하는 위치에 압축 해제합니다.
2. **`install.bat`** 을 더블클릭합니다.
3. 설치가 완료될 때까지 기다립니다. (최대 10~15분)

> Python이 설치되어 있지 않으면 자동으로 설치됩니다.  
> 설치 중 "이 창을 닫고 다시 실행하세요" 메시지가 뜨면, 창을 닫고 `install.bat`을 한 번 더 실행하세요.

---

## 처음 실행 순서

### 1단계 — 데이터 준비 (최초 1회)
**`run.bat`** 을 더블클릭합니다.
- 포켓몬 데이터 자동 수집 (약 10분)
- 오버레이가 뜨면 바로 닫아도 됩니다.

### 2단계 — ROI 설정 (최초 1회)
어떤 화면 영역을 캡처할지 지정합니다.

1. **OBS를 켜고** 닌텐도 스위치 전투 화면을 띄웁니다. (트레이닝 - 배틀 튜토리얼 추천)
2. **`roi_setup.bat`** 을 더블클릭합니다.
3. 안내 메시지를 읽고 아무 키나 누릅니다.
4. **5초 안에 OBS 화면으로 전환**합니다. (스크린샷 자동 촬영)
5. 선택 화면이 나타나면:
   - **초록색 박스**: 내 포켓몬 이름 영역을 드래그합니다.
   - **빨간색 박스**: 상대 포켓몬 이름 영역을 드래그합니다.
6. **Enter** 를 눌러 저장합니다.

### 3단계 — 실행
이후부터는 **`run.bat`** 더블클릭만 하면 됩니다.
- 첫 실행 시 AI 모델 다운로드 (약 5GB, 최초 1회만)
- 오버레이에 **"준비 완료"** 메시지가 뜨면 사용 가능합니다.

---

## 사용 방법

| 동작 | 방법 |
|------|------|
| 스캔 | 오버레이의 **확인** 버튼 클릭 |
| 종료 | 오버레이 우상단 **✕** 버튼 또는 우클릭 → 종료 |

오버레이 표시 예시:
```
상대 포켓몬: 한카리아스 (Spd 102 / 메가 92)
  약점: 얼음×4  드래곤  페어리

내 포켓몬: 윈디 (Spd 95)
  약점: 물  바위  땅
```

## 성능

아군/상대 포켓몬인식 -  선공, 타입상성 출력 까지 '3초'

---

## 문제 해결

| 증상 | 해결 방법 |
|------|-----------|
| 오버레이에 "OCR 초기화 오류" | `install.bat` 재실행 |
| 포켓몬 이름 인식 실패 | ROI 재설정 필요 (`roi_setup.bat`) |
| 스캔 후 인식 실패 시 30초 후 자동 취소됨 | ROI 설정 확인 / 전투 화면에서 스캔했는지 확인 |
| GPU를 인식 못함 | NVIDIA 드라이버 최신 버전으로 업데이트 |

---

## 삭제 방법

**`uninstall.bat`** 을 더블클릭하면 AI 모델 캐시(~10GB)와 패키지(~5GB)가 자동으로 삭제됩니다.
이후 PokeOCR 폴더를 직접 삭제하면 완전히 제거됩니다.
